# 🔐 CTF Toolkit

A modular **Capture The Flag toolkit** written in Python, built for practising ethical hacking and security research. Covers the core tool categories used in CTF competitions.

> ⚠️ For educational purposes and authorised testing only.

## 🧰 Modules

| Module | Description |
|---|---|
| `port_scanner.py` | Async TCP/UDP port scanner with banner grabbing |
| `hash_toolkit.py` | Hash identification, comparison and dictionary attack |
| `web_scanner.py` | Directory fuzzer, header analyser, basic SQLi/XSS detector |
| `encoder.py` | Base64/32/16, ROT13, hex, URL encode/decode |
| `subdomain_enum.py` | DNS subdomain enumeration via wordlist |
| `steganography.py` | LSB stego detection/extraction from images |

## 🚀 Quick Start

```bash
git clone https://github.com/RisKyAyO/ctf-toolkit
cd ctf-toolkit
pip install -r requirements.txt
```
